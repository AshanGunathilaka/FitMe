# FitMe
FitMe Virtual Dressing Room

## Installation

Backend

```bash
cd server
```
```bash
node --watch app
```
```bash
cd server
```
```bash
npx localtunnel --port 8000
```

Frontend

```bash
cd client
```
```bash
npm install 
npm i @reduxjs/toolkit
npm i react-redux
npm i @react-native-async-storage/async-storage
```
```bash
cd client
```
```bash
npx expo start
```
